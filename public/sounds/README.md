# Sound files

Place these audio files here (MP3, WAV, or OGG):

- **drumroll.mp3** – plays when the user clicks Spin
- **cheer.mp3** – plays when the wheel lands (result)
- **fanfare.mp3** – plays when the wheel lands (result)

If your files have different names, rename them or update the paths in `app/components/Wheel.tsx`.
